Oque é JavaScript?

Java Script é uma linguagem de código que atua em contato direto com o cliente ou usuário, é extremamente versátil e compatível com diversos navegadores o que o torna uma das melhores opções para a interação entre cliente e programa, e é a parte do site que geralmente recebe as informações coletadas pelo Front-End.

Oque é Front-End?

Front-End é a parte responsável pela aparência visual e estrutural do site, determinando imagens, posicionamentos e também expansividade. Como é a área visual do site é de extrema importância que muita atenção seja dada a ela visto que o consumidor só terá contato direto com a mesma, é também nesta área que acontece a recepção de dados do programa que então é direcionada ao Back-End.
